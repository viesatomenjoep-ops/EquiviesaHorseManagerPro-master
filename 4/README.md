# Equiviesa - Premium SaaS Landing Page

Premium, high-converting SaaS landing page voor Equiviesa - het ultieme alles-in-één platform voor moderne, professionele paardenstallen.

## 🌍 Multi-Language Support

De landingspagina is beschikbaar in drie talen, allemaal op één pagina:
- **Nederlands** - Volledige Nederlandse versie
- **Español** - Completa versión en español  
- **English** - Full English version

## 🎨 Design

"Apple-meets-Equestrian" design met:
- Premium hunter green en gouden kleurenpalet
- Playfair Display voor koppen, Inter voor body text
- Smooth Motion animaties
- Volledig responsive design

## 🚀 Tech Stack

- **React 18.3.1** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS v4** - Styling
- **Motion** - Animaties
- **Vite** - Build tool
- **Lucide React** - Icons

## 📦 Installation

```bash
pnpm install
```

## 🛠️ Development

```bash
pnpm dev
```

## 🏗️ Build

```bash
pnpm build
```

## 🌐 Deployment

### Vercel (Recommended)

1. Connect repository to Vercel
2. Configure build settings:
   - Build Command: `pnpm build`
   - Output Directory: `dist`
3. Deploy!

### Docker

```bash
docker build -t equiviesa .
docker run -p 3000:80 equiviesa
```

## 🗄️ Supabase Integration

Environment variables needed:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
